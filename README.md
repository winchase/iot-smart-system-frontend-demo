# IoT Monitoring App

Simple React + Vite frontend for the **Monitoring / Dashboard** screen of the Air Quality Analyzer project.

## Run locally

1. Open terminal in this folder
2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Open the local URL shown in terminal

## Current content

- Monitoring page based on the provided mockup
- Header with icons
- Environmental cards
- Two status panels
- Recommendation panel

## Next step
